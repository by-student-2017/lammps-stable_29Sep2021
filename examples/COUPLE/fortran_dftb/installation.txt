## Note
D drive = "/mnt/d". Usually "/mnt/d" => "$HOME" or "~"

## Installation and compiling
1. git clone https://github.com/by-student-2017/lammps-stable_29Sep2021.git
2. cd /mnt/d/lammps-stable_29Sep2021/src
3. make no-message
4. make mpi
5. make mode=lib mpi
6. cd /mnt/d/lammps-stable_29Sep2021/examples/COUPLE/fortran_dftb
7. make

## Usage
1. ./simpleF.x
